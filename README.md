# SkillSphere – Online Learning Platform

A modern online learning platform where users can explore courses, watch lessons, and enroll in skill-based programs like Web Development, Design, Marketing, and more.

## 🌐 Live URL
[https://skill-sphere-gamma.vercel.app](https://skill-sphere-gamma.vercel.app)

## ✨ Key Features
- Browse all available courses
- View top 3 popular courses on home page
- Search courses by title
- User registration and login with email/password
- Google social login
- Protected course details page (login required)
- My Profile page with update feature
- Trending & New Releases section
- Learning Tips section
- Top Instructors section
- Toast notifications for user feedback
- Fully responsive on mobile, tablet, and desktop
- 404 Not Found page

## 📦 NPM Packages Used
- `next` — React framework
- `tailwindcss` — Utility-first CSS
- `daisyui` — Tailwind component library
- `better-auth` — Authentication
- `react-hot-toast` — Toast notifications
- `swiper` — Hero banner slider
- `react-icons` — Icon library
